Hexo插件上的踩坑记录
2019-06-14|坑

Gitalk

这里只列出遇到过的问题，如未找到解决方法可到Gitalk issues 寻找解决方法，或提出issue

评论系统出现 Error:Not found.

可能原因:配置格式不对, 示例如下

client_id: 'XXX'
client_secret: 'XXX'
repo: 'NolotyLand.github.io'
owner: 'NolotyLand'
admin: 'NolotyLand'

评论系统出现 Related issues not found

可能原因:

  1. github仓库 issues未勾选
    仓库->Setting->Features->issues

2.
GitHub Application 里，Authorization callback URL拼写错误，示例如下，注意最后带/

https://nolotyland.github.io/
