Hexo常用命令
2019-06-12|常用

//预览草稿
hexo s –draft

//启动hexo(等价命令, 其实就是命令简写)
hexo s
hexo server

//重新生成hexo
hexo g
hexo generate

//清理已生成的hexo文件
hexo cl
hexo clean

//hexo部署
hexo d
hexo deploy
